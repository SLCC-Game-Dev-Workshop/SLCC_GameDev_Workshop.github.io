// CHAPTER DELIVERABLE

One-Page Game Concept Document

By completing Chapter 1, you'll produce a professional concept document containing:

  • Clear game title and genre
  • One-sentence core gameplay loop
  • 2-3 sentence narrative hook
  • Character profile (3-5 bullets)
  • World description (2-3 sentences)
  • Target audience identification
  • Inspiration list (3-5 works)
